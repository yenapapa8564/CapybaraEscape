구현 범위
클라이언트 (ClientUI)
  • X키 → 오버레이 표시
  • 마우스 드래그로 선 그리기
  • 색상 6가지 / 브러시 3가지
  • 실행취소 (마지막 선 삭제)
  • 완료 시 → GraffitiPlaced 이벤트 전송
서버 (ItemManager)
  • 스프레이 캔 인벤토리 관리
  • 가장 가까운 벽 파트 탐색
  • SurfaceGui 생성 + 스트로크 렌더
  • mazeFolder 하위에 등록 → 스테이지 종료 시 자동 삭제
  • 모든 클라이언트에 낙서 동기화
데이터 (스트로크 포맷)
  • 각 스트로크: 색상 + 굵기 + 포인트 배열
  • 포인트는 0~1 정규화 좌표
  • 스트로크 최대 20개 제한
  • 포인트 간격 최소화로 데이터 경량화
신규 파일/변경
  • Constants: SPRAY_COUNT = 1 추가
  • PlayerManager: 스프레이 인벤토리
  • ItemManager: placeGraffiti() 추가
  • GameManager: GraffitiPlaced 이벤트 연결
  • ClientUI: X키 + 드로잉 UI 추가
  • ReplicatedStorage/Events: GraffitiPlaced
